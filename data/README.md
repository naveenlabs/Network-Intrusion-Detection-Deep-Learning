# Data Directory

## CIC-IDS2017 Dataset

The dataset is too large (717 MB) to host on GitHub.

### Download Instructions

1. Go to: [https://www.kaggle.com/datasets/cicdataset/cicids2017](https://www.kaggle.com/datasets/ericanacletoribeiro/cicids2017-cleaned-and-preprocessed)
2. Download: `cicids2017_cleaned.csv`
3. Place in this `data/` folder
4. Run the notebook

### Dataset Info
- 2.5 million rows
- 52 network flow features
- Binary classification: Benign vs Attack
- Class distribution: 16.89% attacks, 83.11% benign
